## Command Syntax

`/inbox :id :type`

<AccordionGroup>
  <Accordion title="ID">
    Search for a specific case by it's ID. You can see the notification ID in
    your notifications [page](https://connections.squarewebp.app/notifications)
  </Accordion>
  <Accordion title="Type">
    Filter notifications by it's type. Like team invite notifications
  </Accordion>
</AccordionGroup>

<Warning>This command does not have a prefix version</Warning>
