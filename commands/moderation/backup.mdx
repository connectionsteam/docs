<Check>
  Backup command is only available for
  [Premium](https://connections.squarewebp.app/premium) servers
</Check>

# Command Syntax

`/backup load`

## Permission

Only **Physical Owner** of the server can run this command.

<Tip>
  You can use the the backup page in the dashboard of your server to load or
  view the backup of the server
</Tip>
<Warning>This command does not have a prefix version</Warning>
