# Command Syntax

`c.tshoot webhook`

## Webhook argument

Runs a validation in all channels that has a connected connection and is using compact mode.
Check if a webhook exists in the channel and it's usable by the Connections, if not, create one to use in the compact mode.

<Warning>This command does not have a slash command version</Warning>
