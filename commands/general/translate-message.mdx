# Command Syntax

`Translate Message` (It's in message context menu)

<Card title="Explaining" horizontal={true}>
  Translate message context menu lets you translate a message sent by any user
  in a connection channel, the language that will be translated is the language
  set in that current connection or the language of the server
</Card>

![Something](/public/translate-message.png)

Translate Message Context Menu

<Warning>
  You can **not** translate a message that was sent more than 2 weeks ago,
  unless it is frozen
</Warning>
