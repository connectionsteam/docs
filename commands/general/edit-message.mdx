# Command Syntax

`Edit Message` (It's in the message context menu)

<Card title="Explaining" horizontal={true}>
  Edit Message lets you easily edit a message you previously sent in a
  connection channel by opening a modal with the message content
</Card>

![Something](/public/edit-message-raw.png)

Select the command

![Something](/public/edit-message.png)

Edit Message modal

<Warning>
  You can **not** edit a message that was sent more than 2 weeks ago, unless it
  is frozen
</Warning>
